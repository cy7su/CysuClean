## 🚀 Quick Start

<div align="center">

```bash
# 🎯 Clone & Install
git clone https://github.com/cy7su/cysuclean.git
cd cysuclean
npm install

# 🚀 Run Development
npm run tauri dev

# 📦 Build Production
npm run tauri build
```
